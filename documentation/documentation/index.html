1. Installation

To get the latest source JAR file, please refer to the download page. The licencing and technical features are also listed in the download page. To execute the application, JRE 6 or higher needs to be installed on your machine. Prior to the execution, following additional requirements need to be met:

  • NNLS chroma vamp plugin 0.21 or higher needs to be installed on your machine.
  • QM Vamp plugin set 1.7 or higher needs to be installed on your machine.

Please, use the links provided to install all the required software.

2. Basic usage

Run the application by double-clicking the JAR file. In the tabbed environment, choose the desired functionality (chord transition tool, audio analysis tool, or other). In all cases, read through the instructions in the application window from up to down, so all steps are finished in order to get the results.

3. Chord transition tool

Make sure that a MIDI input device is connected to your machine. harmony-analyser application works with MIDI devices connected to the MIDI IN port on your soundcard, or USB port. If you don't have a MIDI input device, you can still use a virtual MIDI keyboard such as vkeybd for Linux (make sure that the raw MIDI ports are available when using virtual keyboards) or just use the text input in Chord transition tool. By hitting capture you can record two chords. Alternative way is to insert the chord in the text feel in absolute format (e.g. C3 E3 G3) or relative format (e.g. C E G). The tool will calculate the harmony complexity of both chords and the transition complexity in between them.

4. Audio analysis tool

Choose the WAV file to analyse and the txt file in which you would want to have the output. Both should be available in the directory where the JAR file is located. To see the available plugins and their settings, click button "List plugins". The analysis can begin and you can proceed by clicking "Extract Chromas", "Extract Chords", "Analyze", or other buttons.

Note: in version 1.0 the process can take a long time, because each chroma is taken into account, without smoothing.
